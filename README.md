## 📝 概要 / Overview

**Solve360R** は，拡張現実（AR）技術を使った革新的な数学パズルゲームです.

## 🎯 遊び方 / How to Play

1. アプリを起動し，平らな床面をデバイスで認識させます
2. AR空間に不等式が表示されます（4方向に配置）
3. 4つの不等式をすべて満たす数字を考えて回答
5. 60秒以内で自己ベストを目指そう！
